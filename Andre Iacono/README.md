# Exercícios de Python - Udemy

Esta pasta foi criada para armazenar meus exercícios, práticas e testes desenvolvidos durante as aulas do curso de Python da Udemy.

## Objetivo

O objetivo desta pasta é registrar minha evolução no aprendizado da linguagem Python, praticando os conceitos vistos em aula e reforçando a lógica de programação por meio de exercícios.

## Conteúdo

Aqui podem estar incluídos exercícios e exemplos relacionados a temas como:

- Sintaxe básica do Python
- Variáveis e tipos de dados
- Operadores
- Estruturas condicionais
- Estruturas de repetição
- Funções
- Listas, tuplas e dicionários
- Programação orientada a objetos
- Manipulação de dados
- Automações e projetos práticos

## Observação

Esta pasta faz parte do meu processo de aprendizado, então os arquivos podem evoluir com o tempo conforme eu avanço no curso e pratico novos conteúdos.
