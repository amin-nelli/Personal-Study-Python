# Personal-Study-Python

Repositório criado para organizar meus estudos em **Python**, reunindo exercícios, desafios, testes práticos e conteúdos de diferentes cursos e referências.

## Objetivo

Este repositório serve como meu espaço de aprendizado e prática, onde concentro:

- exercícios de lógica e programação
- estudos de bibliotecas e frameworks
- desafios técnicos
- anotações e testes de código
- projetos simples para fixação de conteúdo

A ideia é acompanhar minha evolução e manter os estudos organizados por tema, curso ou professor.

## Estrutura do repositório

Atualmente, o repositório está dividido em pastas como:

- **Andre Iacono** → exercícios e conteúdos estudados dessa referência
- **Curso Em Video** → materiais e práticas baseadas no curso
- **Marlon Anesi** → estudos e exercícios relacionados
- **PySide6** → testes e projetos voltados para interface gráfica
- **linkedin challenges** → desafios e práticas voltadas para exercícios e portfólio

## Tecnologias e temas estudados

Ao longo deste repositório, os estudos podem envolver:

- Python
- lógica de programação
- programação orientada a objetos
- manipulação de arquivos
- estruturas de dados
- interfaces gráficas com **PySide6**
- resolução de desafios práticos

## Propósito do projeto

Mais do que guardar arquivos, este repositório representa meu processo de aprendizado em programação, funcionando como:

- histórico de evolução
- ambiente de testes
- base para revisão de conteúdo
- espaço para desenvolvimento de disciplina prática
